Editing a project's files

The Google App Engine Launcher can launch a text editor for editing a project's files.

To edit a project's files:

  1. Select the project in the project list.

  2. Click the Edit button.

    Or: Select the menu option Edit > Open in External Editor...

    Or: Press Command-E.

    The project's application root directory is opened using a text editor.

To open a Finder window for the project's root directory:

  1. Select the project in the project list.

  2. If you have added the "Reveal" button to the toolbar, click the "Reveal" button.

    Or: Select the menu option Edit > Open in Finder...

    Or: Press Shift-Command-F.

    A Finder window opens for the project's root directory.
